error(string, int)

This function is just a part of the Lua standard library, but it is useful enough that I want to mention it here. It outputs the specified string to the screen and stops the script. The int argument is optional. For rapid testing/debugging this is often easier than print() because you don't need to hunt through the huge log file to find it.

error("Track width: " .. trackWidth)

* fif(*, *, *)

Runs the 1st argument through an IF statement, returning the 2nd argument if it is true and the 3rd argument if it is false.

foo = fif(bar == 1, "yes", "no")
-- equivalent code:
if bar == 1 then
    foo = "yes"
else
    foo = "no"
end

table AutomagicTable()

Creates an "automagic table". Attempting to access a sub-table inside the table automatically generates that table.

a = AutomagicTable()
a.b.c.d = "a.b and a.b.c are automatically created"
